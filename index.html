Una ONG ficticia. Un laboratorio real.

TCK ONG es una landing “sandbox” para probar ideas: analítica, experimentación, UX, chatbots y medición. Aquí no salvamos el mundo (todavía), pero sí salvamos dashboards y aprendizajes.

Nota: esta web es un entorno de pruebas para instrumentación (GTM/GA4) y experimentos.

Medición impecable

Eventos claros, naming consistente y una base lista para GTM/GA4. Porque “no medimos” no es una opción.

Experimentación rápida

Un sitio sencillo para lanzar variantes, testear hipótesis y aprender sin depender de veinte repos.

UX + Chatbots

Espacio para integrar chatbots, componentes y micro-interacciones… sin romper nada y con control total.

Pequeños cambios. Grandes aprendizajes.

Esta landing está pensada para iterar: cambiar un titular, mover un CTA, añadir un chatbot o instrumentar un evento. Todo con una base ligera y entendible.

El objetivo no es “la ONG perfecta”, sino un entorno donde practicar buenas decisiones de medición y producto.
